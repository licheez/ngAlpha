About This Demo

This demo showcases the AlphaPrimeScroller component with:

  • ✅ Infinite scrolling with dynamic data loading
  • ✅ Variable-height rows with automatic padding
  • ✅ Progress bar during data loads
  • ✅ Optional fixed row heights
  • ✅ Real-time status updates using signals
  • ✅ 500 sample items with pagination
  • ✅ Configurable page size

Try scrolling to the bottom to load more items, adjusting the configuration options, or refreshing the list to see the scroller in action!
